Full Stack Applications Collection

This repository contains 7 Full Stack and Database-based applications developed as part of Full Stack Development coursework.
The projects demonstrate frontend, backend, and database integration concepts.

A dynamic dashboard application that fetches and sorts data interactively.
Tech: React.js, JavaScript, CSS

2️⃣ Login System

Full stack login system with backend validation and database connectivity.
Tech: HTML, CSS, Node.js, Express, SQL

3️⃣ Order Management (SQL Joins)

Database project demonstrating:

INNER JOIN

LEFT JOIN

RIGHT JOIN

Multi-table joins

4️⃣ Transaction-Based Payment Simulation

Implements SQL transaction control concepts:

BEGIN TRANSACTION

COMMIT

ROLLBACK

Error Handling

5️⃣ Automated Logging (Triggers & Views)

Demonstrates use of:

Database Triggers

Views

Logging mechanisms

6️⃣ Interactive Web Form

Frontend-based interactive form with validation.
Tech: HTML, CSS, JavaScript

7️⃣ Student Registration System

Complete full stack student registration system.
